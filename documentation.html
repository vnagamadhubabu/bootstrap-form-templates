⚙️ How to Use

  1. Unzip the bootstrap-form-templates.zip file.
  2. Open any HTML file (e.g., forms/login.html) in a browser to preview.
  3. Copy the form code and paste it into your existing website or project.
  4. Ensure you have included Bootstrap 5 and FontAwesome CDNs in your project. 
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>
<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.5.2/css/all.min.css">

🎨 Customization

  • Change button colors using Bootstrap classes (e.g., btn-primary, btn-success).
  • Add your own CSS styles inside assets/css/custom.css.
  • Modify icons by replacing <i class="fa fa-user"></i> with other FontAwesome icons.

📱 Responsiveness

All forms are fully responsive using Bootstrap’s grid system. Test on desktop and mobile views.
